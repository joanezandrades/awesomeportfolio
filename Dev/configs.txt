	1) Uma função javascript coleta os dados de um formulário HTML e alimenta um cliente ajax. / Pronto

	2) O ajax envia ao servidor PHP as informações coletadas no passo anterior e aguarda o retorno. / Pronto

	3) Enquanto isso o servidor PHP coleta as informações e faz login no servidor SMTP, solicitando um disparo de e-mail. / Pronto

	4) Se tudo estiver ‘ok’ o servidor SMTP dispara o e-mail e envia uma mensagem de retorno para o servidor PHP. / Pronto

	5) O servidor PHP devolve o retorno para o cliente ajax. / Pronto

	6) O cliente ajax imprime o resultado da requisição no HTML. / Pronto
